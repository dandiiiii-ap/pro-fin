# Pro-Fin Monorepo

## Portal App

### Running Development Server

From the root directory:
```bash
npm run dev:portal
```

Or from the `apps/portal` directory:
```bash
cd apps/portal
npm run dev
```

The application will be available at [http://localhost:5173](http://localhost:5173).

### Building for Production

```bash
cd apps/portal
npm run build
```
